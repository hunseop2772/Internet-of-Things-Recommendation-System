# 🍴🛒 사물인터넷과 추천시스템(식재료 관리,레시피 추천)
개발 기간 : 2019.2.03 ~ 2019.12.10


## 프로젝트 상세 내용 

## 기능
### 👩 사용자 페이지
#### 회원 관리
- 로그인 및 회원가입
- 인물찾기, 비밀번호 찾기
#### 메인 페이지
- 식재료 관리(사용자 냉장고 내 식재료 관리)
- 무게센서를 이용한 식쟈료 관리(내 냉장고에 어떤 재료가 얼마나 있는지)
- 레시피 추천기능(사용자 기반 협업 필터링)
- 레시피 추천기능(콘텐츠 기반 추천)
- 레시피 추천기능(내가 가지고 있는 재료에 따라 추천)
